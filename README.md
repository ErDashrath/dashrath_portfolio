## рҹҡҖ Getting Started

### Prerequisites

- **Node.js** вүҘ 18.x
- **npm** or **yarn**

### Installation

```bash
# Clone the repository
git clone https://github.com/ErDashrath/dashrath_portfolio.git
cd dashrath_portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
